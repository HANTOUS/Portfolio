Project Description

This project is carried out as end of year project, it's a multi-platform event management system in Tunisia which plays the role of an intermediary between those who organize events (concert,camping,hicking...) and those who want to participate in them also it offers the possibility to reserve seats in buses in case the user needs transport to the event and the possibility of renting materials, if necessary, in case of camping for example.

I took part of a team of 6 in the build of this multi-platform(website,Desktop App,Mobile App), we used the SCRUM as a work methodology, we used Symfony 4.4 for the build of the website, JavaFx for the Desktop application and CodeName One for the mobile app consuming RESTful Symfony API.

❮ ❯

What i did

I was responsible of the User Management Module :

  • I created the sign up and log in interfaces using the "AbstractGuardAuthenticator" to grant access for users.
  • I built the profile space where the user can manage his personel inforamtion .
  • I created the space to make claims for the user and their management for the administrator
